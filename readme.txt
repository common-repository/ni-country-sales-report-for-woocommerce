Ni Country Sales Report for WooCommerce provides comprehensive sales reports and analysis based on countries and products.

== Description ==

<p>
The Country Sales Report for WooCommerce plugin offers detailed sales analysis and insights based on countries, including country-specific sales reports, product sales reports, and interactive graphical representations.
</p>
<p>
The plugin includes a user-friendly dashboard with a world map that displays order counts and sales for each country, as well as country-specific sales boxes for quick reference.
</p>
<p>
With the Country Sales Report plugin, you can generate country-wise sales and product-wise sales reports. The reports can be filtered based on the order dates (Today, Yesterday, Last 7 Days, Last 15 Days, Last 30 Days, Last 60 Days, and Last 90 Days), country name, and ordered by billing country, order total, or order count in ascending or descending order.</p>
<p>
Additionally, the plugin provides a comprehensive cross-tab report for products and countries, allowing you to analyze sales patterns. The report can be filtered based on order dates and order billing country.
</p>
<strong>Order and Sorting Options:</strong>
<ul>
	<li>Billing Country: Ascending or Descending order</li>
	<li>Order Total: Ascending or Descending order</li>
	<li>Order Count: Ascending or Descending order</li>
</ul>
<strong>Sales Analysis Indicators for Today:</strong>
<ul>
	<li>Dashboard sales report</li>
	<li>Country-wise sales report</li>
	<li>Country product-wise sales report</li>
	<li>Country-wise order details report</li>
	<li>Bar chart Sales report</li>
	<li>Pie chart Sales report</li>
	<li>Table Country Sales report</li>
</ul>
<strong>Key Features Country Sales Report For WooCommerce:</strong>

== Installation ==

1. Use the WordPress plugin installer or upload the plugin files to the WordPress plugin directory.
2. Activate the "Ni Country Sales Report for WooCommerce" plugin through the WordPress admin panel.
3. Find the "Country Sales" option in the admin menu.
